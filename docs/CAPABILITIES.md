# Consultant Template Capabilities

## Core Capabilities

- **Analytical Review**: Deep analysis of documents, proposals, and artifacts
- **Stakeholder Alignment**: Facilitation of multi-party consensus
- **Risk Assessment**: Identification and mitigation of project risks
- **Strategic Recommendations**: Evidence-based guidance for decisions

## Advisory Mode

This template operates in advisory (aget) mode:
- Read-only by default
- Recommendations require human approval
- Analysis is documented in `.aget/evidence/`
